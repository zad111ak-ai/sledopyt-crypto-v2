# ПОСТ 1: Подготовка Windows — 3 момента, на которых 80% ломают установку

---

**🔧 Серия: "Локальный AI-агент своими руками" — Пост 1/6**

Прежде чем ставить WSL — проверьте эти 3 пункта. Без них установка упадёт.

---

**⚠️ 1. Имя пользователя Windows — ТОЛЬКО латиницей**

Если ваш путь: `C:\Users\Иван\...` — ничего не встанет. WSL и Node.js не переносят кириллицу в путях.

**Решение:** Создайте нового пользователя Windows с английским именем (например `user` или `dev`). Работайте из-под него.

---

**⚠️ 2. Виртуализация в BIOS — должна быть включена**

WSL 2 использует виртуальную машину процессора. Если она выключена — установка упадёт с ошибкой.

**Как включить:**
1. Перезагрузите ПК
2. При загрузке жмите `Del` / `F2` / `F10` (зависит от материнской платы)
3. Ищите раздел `Advanced` → `CPU Configuration`
4. Найдите `Intel VT-x` (Intel) или `AMD-V` / `SVM Mode` (AMD)
5. Переведите в `Enabled`
6. Сохраните (`F10` → `Y`)

---

**⚠️ 3. PowerShell — ТОЛЬКО от имени администратора**

Все команды установки работают только с правами админа.

**Как запустить:**
• Клик правой кнопкой по меню Пуск → "Терминал (Администратор)"
• Или: `Win + X` → "Windows PowerShell (Администратор)"
