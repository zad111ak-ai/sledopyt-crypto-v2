# ПОСТ 3: Установка Hermes Agent — приватный ИИ-помощник за 5 минут

---

**🤖 Серия: "Локальный AI-агент своими руками" — Пост 3/6**

Hermes Agent от Nous Research — это локальный AI-агент, который работает в терминале, читает файлы, запускает команды и управляет вашим ПК. Установка занимает до 2 минут.

---

**Шаг 1. Заходим в Ubuntu**

Открываем PowerShell и пишем:
```
wsl -d Ubuntu-26.04
```

Попадаем в консоль Ubuntu (вместо `C:\>` будет `user@PC:~$`).

---

**Шаг 2. Обновляем систему**

```
sudo apt update && sudo apt upgrade -y
```

Введёт пароль от пользователя Ubuntu (его не видно при вводе — это нормально).

---

**Шаг 3. Устанавливаем зависимости**

```
sudo apt install -y curl git ripgrep ffmpeg build-essential
```

---

**Шаг 4. Ставим Hermes**

```
curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
```

Затем перезагружаем shell:
```
source ~/.bashrc
```

---

**Шаг 5. Проверяем**

```
hermes --version
```

Если видите версию — всё установлено.

---

**Шаг 6. Настраиваем API-ключ**

Чтобы агент общался с мощными моделями, нужен API-ключ.

**Самый универсальный вариант — OpenRouter:**
• Даёт доступ к сотням моделей по одному ключу
• Регистрация: openrouter.ai
• Получаете ключ вида `sk-or-...`

Добавляем ключ:
```
echo "OPENROUTER_API_KEY=sk-or-ваш_ключ" >> ~/.bashrc
source ~/.bashrc
```

---

**Шаг 7. Запускаем**

```
hermes
```

Задайте первый вопрос — агент ответит.

---

**Альтернативы API:**

Если не хотите OpenRouter, Hermes поддерживает 20+ провайдеров:
• Google Gemini (бесплатный тариф)
• DeepSeek (дёшево)
• Anthropic (Claude)
• Ollama (локальные модели, бесплатно)

Настройка: `hermes setup` → интерактивный мастер.
