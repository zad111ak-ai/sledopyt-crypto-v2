# ПОСТ 4: MiMo Code — бесплатный инструмент, который настроит всё за вас

---

**🛠 Серия: "Локальный AI-агент своими руками" — Пост 4/6**

Не хотите копаться в конфигах вручную? MiMo Code сделает всю работу за вас.

MiMo Code — это терминальный AI-ассистент от Xiaomi. Умеет читать и писать код, запускать команды, управлять Git. И сейчас он временно бесплатный.

---

**Зачем он нужен?**

Вы не хотите вручную редактировать конфиги Hermes. Просто ставите MiMo Code и пишете:

*"Настрой Hermes с моим OpenRouter ключом sk-or-ХХХ"*

Он сам отредактирует нужные файлы.

---

**Установка**

В Ubuntu (WSL):
```
curl -fsSL https://mimo.xiaomi.com/install.sh | bash
source ~/.bashrc
```

---

**🎯 Самое неочевидное: копирование в/из MiMo Code**

Это то, на чём зависают 90% новичков. В WSL свои хоткеи:

• **Копировать:** `Ctrl + Shift + C` (не просто Ctrl+C!)
• **Вставить:** `Ctrl + Shift + V`

Альтернатива: правой кнопкой мыши → "Paste" в контекстном меню.

---

**🎯 Лайфхак: пусть MiMo сам подключит API**

1. Запустите `mimo` в терминале
2. Напишите промпт:

*"Установи Hermes Agent и подключи мой OpenRouter API ключ: sk-or-ХХХ. Обнови конфиг."*

3. MiMo сам всё сделает

---

**💰 Про подписки и API**

Для AI-агента нужен API-ключ. Вот варианты от дешёвого к дорогому:

**Бесплатно:**
• Google Gemini — бесплатный тариф, хватает для начала
• Ollama — локальные модели, 0₽, но слабые

**Дёшево:**
• OpenRouter — от $5/мес, доступ к сотням моделей
• DeepSeek — очень дешёвые модели

**Дорого, но мощно:**
• Anthropic Claude — от $20/мес
• OpenAI — от $20/мес

**Совет:** Начните с бесплатного Google Gemini. Потом, когда поймёте что вам нужно — переходите на OpenRouter.
