# ПОСТ 2: Установка Node.js и WSL 2 с Ubuntu за 15 минут

---

**📦 Серия: "Локальный AI-агент своими руками" — Пост 2/6**

Пошаговая инструкция. Просто копируйте команды.

---

**Шаг 1. Ставим Node.js (ДО WSL!)**

Многие ставят WSL первым, а потом мучаются. Сначала Node.

Открываем PowerShell от администратора:

```
winget install OpenJS.NodeJS.LTS
```

После установки перезагружаем ПК (или закройте все терминалы).

Проверяем (в обычном PowerShell):
```
node -v
npm -v
```

Должны появиться версии. Если нет — переустановите.

---

**Шаг 2. Включаем WSL**

Снова PowerShell от администратора:

```
wsl --install
```

Эта команда сама:
• Включит компонент WSL
• Включит виртуальную машину
• Скачает Ubuntu по умолчанию

⚠️ **После завершения — перезагрузите ПК!**

---

**Шаг 3. Ставим Ubuntu 26.04 LTS**

После перезагрузки открываем PowerShell (можно не от админа) и ставим именно LTS-версию — она стабильнее и получает 5 лет поддержки:

```
wsl --install -d Ubuntu-26.04
```

Запустится установка. В конце попросит создать UNIX-пользователя и пароль.

🔑 **Запомните пароль** — он будет нужен часто. При вводе пароль не отображается — это нормально, просто печатайте.

---

**Шаг 4. Проверяем**

В PowerShell:
```
wsl --list --verbose
```

Должно быть: `Ubuntu-26.04` со статусом `Running`.

---

**Готово!** Linux у вас внутри Windows.
